== Description ==

= 5 Minute Setup = 

In just five minutes, you can have brilliant popups on your site.

 = 24 Beautiful Popup Styles = 

A popup for every theme. With 24 different themes to choose from, you can be sure there is a popup that will match your site.

= Animations = 

Draw extra attention to your popups with smooth animations. Choose between different opening and closing animations to increase exposure!

= Choose Where Your Popup Is Displayed = 

Only want your popup in some areas on your site? You can select whether you want them displayed on your posts, pages, or your home page.

= Visual Text Editor = 

Now, no more clunky HTML code to deal with on your popups. Using the WordPress built-in WYSIWYG editor, you can now just worry about what you want to say, and not how to display it.

= Separate Your Code =

WYSIWYG editor drive you up the walls when inserting code? Now keep your code separate with a distinct code box. Simply place code snippets in this box and everything is kept separate.

= Stop Bugging Return Visitors =

Don’t want your return visitors to see this box every time they go back? Don’t want your popup to show on every post people read? Now you can set how often a person sees your popup.

= Stable and 3.0 Compatible = 

No more worrying whether or not the latest version of wordpress will break your plugin. Tested on multiple versions on WordPress with different configuration options, you can rest assured it will 

work right every time.

== Installation ==


= Wordpress Installer = 
1. After you have downloaded your file, log into your WordPress Admin Section.
1. Go to Plugins-> Add New.
1. Go to the Upload Link.
1. Navigate to your file.
1. Click on the Upload Button.
1. After it is has installed, click on the Activation Link.

= Manual Installation =
1. Upload `cool-ryan-easy-popups` plugin folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
